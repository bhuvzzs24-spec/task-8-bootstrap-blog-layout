# Task 8 – Bootstrap Blog Layout

This project is a responsive blog layout created using **Bootstrap 5**.

## Objective

The goal of this task is to build a blog webpage using Bootstrap components such as a navbar, cards, and footer while using the Bootstrap grid system for responsiveness.

## Features

* Responsive navigation bar
* Blog cards with image, title, description and button
* Bootstrap grid layout
* Footer section with icons
* Mobile responsive design

## Technologies Used

* HTML
* Bootstrap 5 (CDN)

## Bootstrap Concepts Used

* Navbar
* Grid System
* Cards
* Utility Classes
* Responsive Layout
